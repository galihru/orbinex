Chapter 3

Scientific Formulations and Their Scala Logic

Every formula below is mapped to practical computation flow and tested in terminal-oriented examples.

Core Equations

F = G * (m1 * m2) / r^2
v = sqrt((G * M) / r)
T = 2 * pi * sqrt(a^3 / (G * M))
a = cbrt((G * M * T^2) / (4 * pi^2))

Implementation Pattern

val mu = G * primaryMassKg
val speed = math.sqrt(mu / orbitalRadiusMeters)
val period = 2.0 * math.Pi * math.sqrt(math.pow(axis, 3) / mu)

Scala Logic Card

Formula to code mapping 
val G = 6.67430e-11
val M = 1.98847e30
val r = 1.495978707e11

val speed = math.sqrt((G * M) / r)
println(f"Speed: $speed%.2f m/s")

val period = 2.0 * math.Pi * math.sqrt(math.pow(r, 3) / (G * M))
println(f"Period: $period%.2f s")
