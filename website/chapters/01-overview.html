Chapter 1

Scientific Overview and Module Architecture

Orbinex is structured for scientific computing workflows in Scala, especially astronomy simulations that require reproducible formulas and configurable data ingestion.

Module Roles

  • orbinex-core: deterministic formulas, constants, coordinate transforms.
  • orbinex-fetch: dynamic HTTP ingestion for exoplanet and agency data.
  • orbinex: unified facade API for downstream usage.

Scientific Scope

  • Newtonian gravitational mechanics.
  • Orbital period and semi-major axis conversion.
  • RA/Dec coordinate projection into Cartesian space.
  • Deterministic fallback estimations for incomplete datasets.

              Single-call formula usage
              

                
                
                
              

            

            
import io.github.galihru.orbinex.Orbinex

val v = Orbinex.formulas.circularOrbitSpeed(
  primaryMassKg = Orbinex.constants.SolarMass,
  orbitalRadiusMeters = Orbinex.constants.AU
)

println(f"v(1 AU) = $v%.2f m/s")
